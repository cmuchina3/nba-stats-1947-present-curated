# Data Dictionary

This document provides definitions for commonly used variables across the NBA/ABA/BAA statistics dataset.

## Common Variables

| Variable | Description |
|-----------|------------|
| Player | Full name of player |
| Season | NBA season year (ending year format) |
| Team | Team abbreviation |
| Age | Player age during the season |
| G | Games played |
| GS | Games started |
| MP | Minutes played |

---

## Shooting Statistics

| Variable | Description |
|-----------|------------|
| FG | Field Goals Made |
| FGA | Field Goal Attempts |
| FG% | Field Goal Percentage |
| 3P | 3-Point Field Goals Made |
| 3PA | 3-Point Field Goal Attempts |
| FT | Free Throws Made |
| FTA | Free Throw Attempts |
| FT% | Free Throw Percentage |

---

## Advanced Metrics

| Variable | Description |
|-----------|------------|
| PER | Player Efficiency Rating |
| TS% | True Shooting Percentage |
| WS | Win Shares |
| WS/48 | Win Shares per 48 Minutes |
| BPM | Box Plus/Minus |
| VORP | Value Over Replacement Player |

---

## Possession-Based Statistics

| Variable | Description |
|-----------|------------|
| Per 100 Poss | Statistics normalized per 100 possessions |
| Per 36 | Statistics normalized per 36 minutes |

---

## Notes

- Statistical definitions may vary historically as tracking methodologies evolved.
- Advanced metrics were introduced in later eras and may not exist for early seasons.
- Team abbreviations reflect historical naming conventions.
