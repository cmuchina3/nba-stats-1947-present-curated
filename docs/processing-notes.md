# Processing Notes

## Source

Original dataset obtained from Kaggle:
“NBA-ABA-BAA Stats (1947–Present)”

Data originally compiled from Basketball Reference.

---

## Curation Actions Performed

The following steps were taken to prepare this dataset for repository-style publication:

- Organized files into structured directory layout
- Standardized file naming conventions (snake_case)
- Added structured DataCite XML metadata
- Created repository-level README documentation
- Added data dictionary and preservation notes
- Documented licensing and provenance considerations

---

## Data Integrity

- No statistical values were modified.
- No rows or records were removed.
- No cleaning or transformation of numerical values was performed.

All CSV files represent the original Kaggle dataset contents.

---

## Known Limitations

- Column-level documentation was not provided by the original source.
- Historical metric definitions may vary across eras.
- Advanced metrics do not exist for early seasons.

Users should interpret statistics within historical context.
